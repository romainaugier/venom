# venom

venom aims to be python compiler and toolchain but is mostly a playground for me to test and learn stuff about compilers. I didn't wanted to invent a new language, and I hope one day it will be useful to some people. For now, nothing works because it is in progress. It also aims to be dependency-free (except libromano that provides a lot of utilities that would be redundant to code again), very fast to build, and very fast to execute.
