# venom

venom aims to be Python compiler and toolchain but is mostly a playground for me to test and learn stuff about compilers. I didn't wanted to invent a new language, and I hope one day it will be useful to some people (god knows how much energy and time is wasted with Python slow execution). 
Note that for now, nothing works in a stable way because the development is in progress; once I'll get something working well, different branches will be created to ease the track of development.
It also aims to be dependency-free (except libromano that provides a lot of utilities that would be redundant to code again), very fast to build, and very fast to execute.
